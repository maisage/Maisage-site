How to use these learn pages

These are meant to be skimmed and used as conversation fuel — not memorized.

1. Pick one or two topics

Start with whatever is causing the most stress: housing, money, meds, or “what happens next.” Read only those pages first.

2. Turn notes into questions

For each page, write down a few questions you want to bring to doctors, social workers, facilities, or financial professionals.

3. Pair with tools & services

When ready, use the Tools page to compare scenarios, or the Services page if you want structured help walking through options.
